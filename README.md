# 프로젝트 소개

'[NEWNEEK] (https://newneek.co/) 뉴스 구독 홈페이지인 '뉴닉'의 리뉴얼을 진행했습니다.조원 정다현님, 이상희님, 주자명님과 기획단계를 함께 했고 시안 디자인 및 퍼블리싱은 개인적으로 진행했습니다.

리뉴얼을 진행함에 앞서 본 사이트를 아래와 같이 분석했습니다.

  <hr/>

- ### 목표

1. 반응형 웹으로 구현할 것
2. 서비스 주 타겟층을 고려한 역동적인 디자인을 창작할 것
3. 서비스를 메인페이지에 나열하여 사용자의 서비스 접근성을 높일 수 있게 할 것
<hr/>

- ### 시안Mood / 벤치마킹 요소

1. 디자인 Mood image를 선정 후 구체적인 컨셉을 설정하였습니다.
